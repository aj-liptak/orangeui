detach

(

  • [call]
)

Inherited from Events: api/orange.js:271

Detaches listeners from an object. Specifying the event and function parameters will remove that specific listener, while specifying just the event name will remove all listeners for that event. No parameters will remove all bound listeners to the object.

Parameters:

  • [ev] String optional

    The optional name of the event to unbind.

  • [call] Function optional

    The option listener to unbind.
