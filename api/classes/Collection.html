Filters a collection on a given keyword or set of keywords. The filters will be applied to a specific field, or all fields if none are specified. The filter can also be applied to the existing already filtered list. By default, the list will filter from its unfiltered state.

Parameters:

  • keyword String | Array

    The keyword or array of keywords to filter on.

  • [field] String optional

    The field to filter on, or none if testing all fields.

  • [existing] Bool optional

    Whether or not to start the filter from the entire set.
